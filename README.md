# 🗄️ SQL Learning Journey

A personal repository documenting my progress learning SQL. All practice is done using my local **`practice_sql`** database in SQL Server Management Studio (SSMS).

---

## 📁 Folder Structure

```
sql-learning-journey/
├── basics/              # Core SQL syntax (SELECT, WHERE, ORDER BY, etc.)
├── joins/               # INNER, LEFT, RIGHT, FULL joins
├── aggregations/        # GROUP BY, HAVING, COUNT, SUM, AVG, etc.
├── subqueries/          # Nested queries and CTEs
├── dml/                 # INSERT, UPDATE, DELETE, MERGE
├── ddl/                 # CREATE, ALTER, DROP tables/schemas
├── stored-procedures/   # Stored procedures and functions
├── views/               # Creating and managing views
├── indexes/             # Index creation and performance tips
├── projects/            # Mini end-to-end practice projects
└── notes/               # Concept notes and quick reference sheets
```

---

## 🛠️ Setup

- **Database**: `practice_sql` (local SQL Server instance)
- **Tool**: SQL Server Management Studio (SSMS)
- **SQL Dialect**: T-SQL (Transact-SQL)
